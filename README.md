# dotfiles

## Installation

```sh
curl -fsSL https://raw.githubusercontent.com/o8n/dotfiles/master/install.sh | bash
```

Or manually:

```sh
git clone https://github.com/o8n/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Contents

- `.zshrc` - Zsh configuration
- `.vimrc` - Vim configuration
- `.tmux.conf` - tmux configuration
- `nvim/` - Neovim configuration (lazy.nvim)
